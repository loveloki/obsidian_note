# Javascript 心智模型

## 类型

### 原始类型 和对象

 > 原始类型无法更改，是基石。（是值）
 > 
 > 对象可以被修改。（是引用）
 >  
 >  传递参数的时候，是什么就传递什么，所以传递原始类型不会有问题，传递对象会存在修改后一起被改变的问题（因为指向同一个引用）

**原始类型**

原始类型为什么可以调用方法？因为包含了隐式转换，转换为了相应的对象，进行了临时包装。

- [[undefined]]
- [[null]]：虽然 `typeof null` 输出是 `object` ，但是这是一个历史遗漏问题，实际上不是对象。
- [[string]]
- [[number]]：只有浮点类型，没有整数类型。
- [[boolean]]
- [[symbol]]
- [[bigint]]

**对象类型**

- [[Object]]：除了原始类型都是对象
